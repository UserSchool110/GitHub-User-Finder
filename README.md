## Описание программы

**GitHub User Finder** — это GUI-приложение для поиска пользователей GitHub, просмотра информации о них и сохранения избранных пользователей в локальный файл. Приложение использует официальное GitHub API для поиска и получения данных.

### Возможности:  
- ✅ Поиск пользователей GitHub по логину  
- ✅ Отображение списка найденных пользователей с указанием логина и имени  
- ✅ Добавление пользователей в избранное  
- ✅ Сохранение избранных пользователей в JSON-файл  
- ✅ Загрузка избранного из JSON-файла при запуске  
- ✅ Удаление пользователей из избранного  
- ✅ Проверка пустого поля

## Инструкция по запуску

### Требования:  
- Python 3.6 или выше  
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)  
- Библиотека `requests` (для работы с GitHub API)

### Установка зависимостей:
```bash
pip install requests
```

### Запуск:  
1. Откройте терминал или командную строку  
2. Перейдите в директорию проекта:  
   ```bash
   cd github-user-finder
   ```  
3. Запустите приложение:  
   ```bash
   python github_user_finder.py
   ```

## Примеры использования

### Поиск пользователя:  
1. Введите логин пользователя в поле поиска (например, `octocat`)  
2. Нажмите кнопку **«Поиск»** или клавишу **Enter**  
3. В левом списке отобразятся найденные пользователи с указанием логина и имени

### Добавление в избранное:  
1. Выберите пользователя из списка результатов поиска  
2. Нажмите кнопку **«Добавить в избранное»**  
3. Пользователь появится в правом списке «Избранное»

### Удаление из избранного:  
1. Выберите пользователя в правом списке «Избранное»  
2. Нажмите кнопку **«Удалить»**  
3. Пользователь будет удалён из избранного

### Сохранение данных:  
- Данные избранных пользователей **сохраняются автоматически** в файл `favorites.json` при каждом добавлении или удалении  
- При следующем запуске программы избранные пользователи автоматически загружаются

### Сброс результатов:  
- Нажмите кнопку **«Очистить»**, чтобы очистить поле поиска и список результатов

## Валидация данных

Приложение проверяет:  
- Поле поиска не должно быть пустым (иначе выводится предупреждение)  
- При добавлении в избранное необходимо выбрать пользователя из списка результатов  
- При удалении из избранного необходимо выбрать пользователя из списка избранного  
- Повторное добавление пользователя в избранное запрещено
